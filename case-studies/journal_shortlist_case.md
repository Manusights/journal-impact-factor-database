# Case Study: Building a Journal Shortlist

## Scenario
A team has a translational oncology manuscript and needs a target journal plus two backups.

## Input state
- 28 candidate journals
- no ranking logic
- mixed evidence strength across figures

## Workflow used
1. Applied scorecard (scope, claim fit, operational fit, audience fit)
2. Removed journals with obvious scope mismatch
3. Performed claim-to-figure review
4. Finalized top 3 targets

## Output
- Primary target selected with explicit rationale
- Two backup journals with lower mismatch risk
- Submission plan documented in tracker

## What improved
- Faster internal alignment
- Fewer speculative submissions
- Better cover letter positioning
