# Data License and Usage Notes

This repository is designed for open, reproducible workflows around journal metadata.

## What this repo does include
- Open schemas and templates
- Publicly reproducible transformation patterns
- Example records for tooling development

## What this repo does not include
- Proprietary Clarivate raw exports
- Any licensed JCR file redistribution

## How to use with licensed data
If your institution has JCR access, you can map licensed fields into the public schema in this repository for internal analysis.

## Suggested citation format
If this repository helps your workflow, please cite it as:

> Manusights. Journal Impact Factor Database (Research Utility). GitHub repository. https://github.com/Manusights/journal-impact-factor-database
