# How to Build a Journal Shortlist

This playbook gives a reproducible way to move from a large journal set to a submission shortlist.

## Step 1. Define hard constraints
- Scope fit for your paper topic
- Article type accepted by journal
- Open access requirement from funder
- Maximum acceptable review time

## Step 2. Define scoring criteria
Use a weighted score from 0 to 100.

- Scope fit: 35
- Evidence and claim match to journal standard: 25
- Operational fit (timeline, transfer policy): 20
- Visibility goals: 20

## Step 3. Create a first pass list
Start with 20 to 40 journals in your field, then remove obvious misfits.

## Step 4. Run a structured scoring pass
For each journal, score each criterion and keep notes for why.

## Step 5. Select top 3
- Primary target
- Realistic target
- Fast fallback target

## Output format
Include one line per journal:
- Journal
- Total score
- Top reason to submit
- Main risk
