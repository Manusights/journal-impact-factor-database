# Common Data Errors in Journal Tables

- Mixed ISSN formats
- Duplicate journals with slight name changes
- Quartile values outside expected set
- Non-numeric impact factor values
- Outdated submission links


## Why this matters
A consistent process reduces avoidable journal mismatch and rework.
